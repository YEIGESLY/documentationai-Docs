---
title: "WordPress + HTTP API"
description: "Sincroniza usuarios de WordPress mediante HTTP API."
---

<Callout kind="info">Conecta tu WordPress con E-SMART360 usando la API HTTP para sincronizar usuarios y datos.</Callout>
<Steps>
  <Step title="Obtén API Key">En E-SMART360, genera una API Key.</Step>
  <Step title="Crea webhooks en WordPress">Usa un plugin o código personalizado para enviar datos de usuarios nuevos.</Step>
  <Step title="Configura flujo en E-SMART360">Disparador webhook entrante para recibir datos.</Step>
  <Step title="Procesa datos">Crea contactos, envía mensajes de bienvenida, asigna etiquetas.</Step>
</Steps>
<Callout kind="tip">Alternativa: usa el plugin de WordPress de E-SMART360 si prefieres no manejar webhooks manualmente.</Callout>
