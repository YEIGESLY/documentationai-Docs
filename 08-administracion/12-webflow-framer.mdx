<Callout kind="info">Integra el Webchat de E-SMART360 en sitios construidos con Webflow o Framer.</Callout>
<Steps>
  <Step title="Obtén código del widget">En E-SMART360, copia el código JavaScript del widget.</Step>
  <Step title="Webflow">Agrega el código antes del &lt;/head&gt; en Custom Code del sitio.</Step>
  <Step title="Framer">Agrega el código en Settings > Custom Code.</Step>
  <Step title="Widget activo">El chat aparece en tu sitio. Configura apariencia desde E-SMART360.</Step>
</Steps>
