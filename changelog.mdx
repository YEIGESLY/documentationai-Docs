## Upgrade Process

Follow these steps to update your E-Smart360 implementation safely.

<Steps>
  <Step title="Check Compatibility" icon="check-circle">

    Review your current version and changelog for breaking changes.

    <Callout kind="alert">
      Backup your configuration before proceeding.
    </Callout>

  </Step>
  <Step title="Update Package" icon="download">

    Use your package manager to install the latest version.

    <CodeGroup tabs="npm,yarn">
      ````bash
      npm install e-smart360@latest
      ````

      ````bash
      yarn add e-smart360@latest
      ````
    </CodeGroup>

  </Step>
  <Step title="Reconfigure" icon="settings">

    Update your API endpoint to `https://api.example.com/v1` and regenerate keys.

    <ParamField path="api_key" param-type="string" required="true">
      Your new E-Smart360 `{API_KEY}`.
    </ParamField>

    <ParamField header="Authorization" param-type="string" required="true">
      Bearer `{YOUR_TOKEN}`.
    </ParamField>

  </Step>
  <Step title="Test Integration" icon="zap">

    Verify Meta API connections and AI chat flows.

  </Step>
</Steps>
