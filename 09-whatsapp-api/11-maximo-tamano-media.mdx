description: "Límites de tamaño y formato para archivos multimedia en WhatsApp."
---

<Callout kind="info">WhatsApp tiene límites específicos de tamaño y formato para cada tipo de archivo multimedia.</Callout>

## Límites por Tipo de Archivo
| Tipo | Tamaño máximo | Formatos soportados |
|------|---------------|---------------------|
| Imagen | 5 MB | JPG, PNG, GIF, WEBP |
| Video | 16 MB | MP4, 3GPP |
| Audio | 16 MB | AAC, MP4, AMR, MP3, OGG |
| Documento | 100 MB | PDF, DOCX, XLSX, PPTX, TXT |

## Recomendaciones
- **Imágenes**: 800x800 px mínimo, formato JPG o PNG.
- **Videos**: 30 segundos máximo para mejor experiencia.
- **Documentos**: PDF es el formato más universal.
- **Audio**: AAC para mejor calidad con menor tamaño.

<Callout kind="warning">Si envías un archivo que excede el límite, WhatsApp devolverá un error y el mensaje no se entregará. Comprime los archivos antes de enviarlos.</Callout>
