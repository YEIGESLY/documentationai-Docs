<Callout kind="info">Soluciones prácticas para los errores más comunes al usar WhatsApp Business API con E-SMART360.</Callout>

## Error 401 - No autorizado
**Solución**: Regenera tu API Key en Administración > Seguridad > API Keys y actualízala en la integración.

## Error 470 - Fuera de ventana 24h
**Solución**: Usa una plantilla aprobada (marketing o utilidad) para contactar al cliente. No puedes enviar mensajes de servicio libres fuera de la ventana.

## Error 481 - Plantilla rechazada
**Solución**: 
1. Lee el motivo del rechazo en la notificación.
2. Corrige el contenido según las políticas de Meta.
3. Vuelve a enviar la plantilla a revisión.

## Error 471 - Límite alcanzado
**Solución**:
1. Espera al día siguiente para que se reinicie el contador.
2. Mejora tu calificación de calidad para aumentar el límite.
3. Revisa que no estés enviando mensajes duplicados.

## Error 472 - Frecuencia excedida
**Solución**: Reduce la frecuencia de mensajes al mismo contacto. Respeta el frequency capping (máx. 1 marketing/día).

<Callout kind="tip">Si el error persiste después de intentar las soluciones, contacta a soporte de E-SMART360 para asistencia personalizada.</Callout>
