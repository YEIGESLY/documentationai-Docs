<Callout kind="info">
  La API de WhatsApp Business es la solución oficial de Meta para que empresas envíen y reciban mensajes de forma programática, escalable y profesional, muy superior a la aplicación normal de WhatsApp.
</Callout>

## ¿Cómo funciona?

La API de WhatsApp Business (Cloud API) funciona a través de un modelo de **conversaciones** en lugar de mensajes individuales. Esto significa que:

1. **Conversaciones iniciadas por la empresa** — Requieren usar una **plantilla de mensaje** aprobada por Meta.
2. **Conversaciones iniciadas por el cliente** — Abres una ventana de 24 horas para responder libremente.
3. **Conversaciones de servicio** — Relacionadas con pedidos, soporte y seguimiento.

<Callout kind="warning">
  La ventana de 24 horas es importante: si respondes después de ese tiempo, deberás usar una plantilla de mensaje para retomar la conversación.
</Callout>

## Beneficios para tu negocio

- **Escalabilidad** — Atención masiva sin perder calidad
- **Profesionalismo** — Perfil de negocio verificado con palomita verde
- **Automatización** — Chatbots que trabajan 24/7
- **Segmentación** — Campañas dirigidas a públicos específicos
- **Analytics** — Métricas de entrega, lectura y respuesta
- **Integración** — Conecta con tu infraestructura actual

## ¿Qué necesitas para usarla?

1. Un **número de teléfono** que no esté registrado en WhatsApp
2. Una **cuenta de Facebook Business Manager**
3. **Verificación de negocio** ante Meta
4. Una plataforma como **E-SMART360** para gestionarla
