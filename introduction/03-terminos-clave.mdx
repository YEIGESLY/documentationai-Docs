## Términos Esenciales

| Término | Definición |
|---------|-----------|
| **BSP** (Business Solution Provider) | Proveedor oficial de soluciones de WhatsApp Business API. E-SMART360 opera como BSP. |
| **Cloud API** | La API de WhatsApp Business alojada en la nube de Meta. No requiere servidores propios. |
| **WABA** (WhatsApp Business Account) | Tu cuenta de WhatsApp Business dentro de Meta. Cada negocio puede tener una o varias. |
| **Business Manager** | Centro de administración de Meta para gestionar cuentas, usuarios y permisos. |
| **Embedded Signup** | Flujo de registro integrado que conecta tu WhatsApp con E-SMART360 en pocos clics. |
| **Phone Number ID** | Identificador único de tu número de teléfono en la API de Meta. |
| **Message Template** | Plantilla de mensaje preaprobada por Meta para iniciar conversaciones. |
| **Conversation** | Unidad de facturación de Meta. Ventana de 24 horas desde el primer mensaje. |
| **Quality Rating** | Calificación de calidad de tu número (Verde/Amarillo/Rojo) basada en la interacción con usuarios. |
| **Messaging Tier** | Nivel de mensajería que determina cuántos mensajes puedes enviar por día. |

## Conceptos de Automatización

<Columns cols="2">
  <Card title="Flow Builder">
    Editor visual para crear flujos de conversación: disparadores, acciones, condiciones y variables. No requiere programación.
  </Card>
  <Card title="Disparador (Trigger)">
    Evento que inicia un flujo. Ejemplos: mensaje entrante, palabra clave, formulario enviado, compra realizada.
  </Card>
  <Card title="Keyword / Palabra clave">
    Palabra o frase que el usuario escribe y que activa una respuesta automática específica.
  </Card>
  <Card title="Broadcasting">
    Envío masivo de mensajes a una lista de contactos utilizando plantillas aprobadas.
  </Card>
</Columns>

## Términos de Marketing

- **Plantilla de marketing** — Para promociones y ofertas. Tarifa más alta.
- **Plantilla de utilidad** — Notificaciones de pedidos, facturas, alertas. Tarifa menor.
- **Plantilla de servicio** — Respuestas a consultas de clientes dentro de la ventana de 24 horas. Sin costo adicional.
- **Plantilla de autenticación** — Códigos OTP y verificación en dos pasos.
- **Segmentación** — División de contactos por etiquetas, campos personalizados o comportamiento.

## Términos de la Bandeja Compartida

| Término | Definición |
|---------|-----------|
| **Live Chat** | Bandeja de entrada compartida donde los agentes ven y responden conversaciones en tiempo real. |
| **Handover** | Transferencia de una conversación de un bot a un agente humano. |
| **Nota interna** | Mensaje visible solo para agentes, no para el cliente. |
| **Firma de mensaje** | Texto personalizado que identifica al agente que responde. |
