---
title: Notas Internas en Conversaciones
description: Agrega notas privadas visibles solo para el equipo.
---
Las notas internas permiten a los agentes dejar comentarios privados sobre una conversación, visibles solo para el equipo.

## Usos de Notas Internas

- Registrar información relevante del cliente.

- Dejar instrucciones para el siguiente turno.

- Anotar detalles de una gestión en proceso.

- Señalar problemas recurrentes para seguimiento.

## Cómo Usarlas

Usa notas para compartir información útil entre turnos: "Cliente llamó antes. Prefiere que le hablemos de tú".
