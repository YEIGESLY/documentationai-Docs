<Callout kind="info">Make (antes Integromat) y Pabbly Connect son alternativas a Zapier para integraciones más complejas.</Callout>

## Make
<Steps>
  <Step title="Busca E-SMART360">En Make, busca el módulo de E-SMART360.</Step>
  <Step title="Escenarios">Crea escenarios visuales con múltiples pasos y condiciones.</Step>
  <Step title="Ventaja">Múltiples rutas, filtros, agregadores. Más control que Zapier.</Step>
</Steps>

## Pabbly Connect
<Steps>
  <Step title="Busca el módulo">En Pabbly Connect, usa HTTP API de E-SMART360.</Step>
  <Step title="Configura webhooks">Recibe datos desde E-SMART360 o envía mensajes.</Step>
  <Step title="Ventaja">Plan gratuito generoso + operaciones ilimitadas en plan de paga.</Step>
</Steps>

<Callout kind="tip">Make es ideal para flujos complejos con transformaciones de datos. Pabbly es mejor para presupuestos ajustados.</Callout>
