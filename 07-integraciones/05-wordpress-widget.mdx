---
title: "Widget de Chat para WordPress"
description: "Agrega el chat de E-SMART360 a tu sitio WordPress."
---

<Callout kind="info">El widget de chat para WordPress integra el Webchat de E-SMART360 directamente en tu sitio web.</Callout>
<Steps>
  <Step title="Instala el plugin">Descarga el plugin de E-SMART360 desde el repositorio de WordPress.</Step>
  <Step title="Activa y configura">Ingresa tu API Key y personaliza colores y posición.</Step>
  <Step title="Personaliza mensajes">Saludo de bienvenida, mensaje fuera de línea, colores.</Step>
  <Step title="Widget activo">El chat aparece en tu sitio web. Los visitantes pueden chatear contigo.</Step>
</Steps>
<Columns>
  <Card title="Chat en vivo">Atención en tiempo real desde E-SMART360.</Card>
  <Card title="Chatbot automático">Responde sin agentes disponibles.</Card>
  <Card title="Multicanal">Unifica webchat y WhatsApp.</Card>
</Columns>
