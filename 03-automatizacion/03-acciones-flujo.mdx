<Callout kind="info">Las acciones ejecutan tareas específicas dentro del flujo.</Callout>
## Mensajes
- **Texto**: Con formato, variables ({{nombre}}), emojis.
- **Multimedia**: Imagen 5MB, Video 16MB, Documento 100MB, Audio 16MB.
- **Interactivo**: Botones CTA, listas (10 opc.), botones rápidos (3).
## Lógica
<Columns cols="2">
  <Card title="Condición">If/Else con igual, contiene, mayor/menor que.</Card>
  <Card title="Temporizador">Pausa de segundos a 24h.</Card>
  <Card title="Variable">Modifica valores de contexto.</Card>
  <Card title="Script">JavaScript para lógica avanzada.</Card>
</Columns>
## Integración
- HTTP Request, Webhook saliente, Google Sheets.
## Gestión
- Asignar agente, etiquetar, cerrar conversación.
