<Callout kind="info">Referencia completa de códigos de error de la API de WhatsApp y E-SMART360.</Callout>

## Códigos HTTP
| Código | Significado | Solución |
|--------|-------------|----------|
| 200 | Éxito | — |
| 400 | Petición incorrecta | Revisa formato JSON y campos requeridos |
| 401 | No autorizado | API Key inválida o expirada |
| 403 | Prohibido | Sin permisos para esta acción |
| 404 | No encontrado | Recurso no existe |
| 429 | Demasiadas peticiones | Reduce la frecuencia de llamadas |
| 500 | Error interno | Contacta a soporte |

## Códigos Específicos WhatsApp
| Código | Error |
|--------|-------|
| 130472 | Número inválido |
| 130429 | Límite de rate alcanzado |
| 131000 | Servicio no disponible |
| 131042 | Cuenta en revisión |
| 132001 | Plantilla rechazada |

<Callout kind="info">Puedes consultar el log de errores en tiempo real desde el dashboard de E-SMART360 en Registros > Actividad.</Callout>
